# ACMer-Simulator
XCPC选手模拟器

## 🎮 项目简介

ACMer选手模拟器是一个单页面游戏，让你体验编程竞赛选手的生活！通过分配属性点、参加比赛、解决问题来提升你的ACM技能，与全球玩家竞争排名。

## ✨ 核心功能

### 🎯 游戏控制
- **开始游戏**: 开启你的ACMer之旅
- **暂停/继续**: 灵活控制游戏进度
- **重置游戏**: 重新开始全新的挑战

### 📊 属性系统
分配15个初始属性点到通用能力和专业领域：

#### 通用属性
- 💻 **编程能力**: 影响代码质量和解题效率
- 🧮 **算法思维**: 决定算法理解和应用能力
- 🏃 **速度**: 提升做题和编码速度
- 🧘 **抗压能力**: 应对比赛压力的关键
- 🤝 **团队协作**: 团队赛中的重要因素
- 🌐 **英语能力**: 理解英文题目和文档的能力

#### 专业属性
- 📐 **数学**: 数学建模和推导能力
- 🔄 **动态规划**: 动态规划问题的解决能力
- 🕸️ **图论**: 图论算法的理解和应用
- 🗂️ **数据结构**: 数据结构的掌握和运用

### 🏆 全球统计
- 📈 实时全球玩家统计
- 🥇 Top 10排行榜
- 📊 平均分数和游戏时长
- 👤 个人成绩追踪

### 🎲 游戏机制
- 每5天自动参加一次比赛
- 每天根据属性解决编程问题
- 属性影响比赛得分和解题成功率
- 动态排名系统

## 🚀 快速开始

### 本地开发
```bash
# 克隆仓库
git clone https://github.com/a48zhang/ACMer-Simulator.git

# 进入目录
cd ACMer-Simulator

# 安装依赖
npm install

# 启动开发服务器
npm run dev
# 然后访问 http://localhost:5173
```

### 生产构建
```bash
# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## 🎯 游戏玩法

1. **分配属性点**: 在游戏开始前，根据你的策略分配15个属性点到通用属性和专业属性
2. **开始游戏**: 点击"开始游戏"按钮启动模拟
3. **自动进行**: 游戏会自动模拟你的ACM生涯，参加比赛、解题
4. **观察成长**: 关注你的分数、解题数、比赛次数和排名变化
5. **冲击榜首**: 尝试不同的属性分配策略，争取登上排行榜！
